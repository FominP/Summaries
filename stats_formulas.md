# Формулы для собеседований по математической статистике
Теорема Байеса: $P(A) * P(B|A) = P(B) * P(A|B) = P(A \cap B)$   
Дисперсия в равномерном и нормальном распределениях = $ Var(X) = M((X- \mu )^2) = M((X^2)_{ср} - (X_{ср}) ^2) $
 
### Размещения и перестановки
$P_n^k = \frac{n!}{(n-k)!}$ (Разместить 5 людей на 3 стульях)  
$C_n^k = \binom{n}{k} = \frac{n!}{k!(n-k)!}$ (Выбрать 3 человек из 5)  
## Формулы для связанных распределений
**Распределение Бернулли** - заданы только вероятность успеха и неудачи (частный случай биномиального распределения для n=1)  
**Биномиальное распределение** - задано то, что в Бернулли + количество экспериментов  
**Геометрическое распределение** - "сколько испытаний нужно провести, чтобы получить первый успех?" в условиях биномиального распределения

| Распределение  | Var(X)            | E(x)          |
| -------------- | ----------------- | ------------- |
| Бернулли       | $p(1-p)$          | $p$           |
| Биномиальное   | $np(1-p)$         | $np$          |
| Геометрическое | $\frac{1-p}{p^2}$ | $\frac{1}{p}$ |

Вероятность в биномиальном распределении = $P_n(k) = C_k^n * p^k * q^{n-k}$  

## Статистические критерии
В основе статистических критериев лежит идея сравнивать наблюдаемое значение с теоретическим распределением таких значений  
**Z-оценка** = $Z = \frac{X - \mu}{\sigma}$ 

### Примеры критериев
**Z-критерий для сравнения двух пропорций**:  
$$Z = \frac{\hat{p}_1 - \hat{p}_2}{\sqrt{\hat{p}(1-\hat{p})\left( \frac{1}{n_1} + \frac{1}{n_2} \right)}}, \quad \hat{p} = \frac{x_1 + x_2}{n_1 + n_2}$$


**t-критерий Стьюдента** (для двух независимых выборок с равными дисперсиями):  
$$t = \frac{\bar{X}_1 - \bar{X}_2}{s_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}, \quad s_p = \sqrt{\frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}}$$

**t-критерий Уэлча** (для неравных дисперсий, поправка на степени свободы):  
$$t' = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}, \quad \nu \approx \frac{\left( \frac{s_1^2}{n_1} + \frac{s_2^2}{n_2} \right)^2}{\frac{(s_1^2/n_1)^2}{n_1-1} + \frac{(s_2^2/n_2)^2}{n_2-1}}$$


**Критерий Манна-Уитни** (U-тест, для двух независимых выборок):  
$$U = \min(U_1, U_2), \quad U_1 = n_1 n_2 + \frac{n_1(n_1+1)}{2} - R_1, \quad U_2 = n_1 n_2 - U_1$$
где $R_1$ — сумма рангов первой выборки.

**Хи-квадрат Пирсона** = $\chi^2 = \sum_{i=1}^{k} \frac{(Fact_i - Exp_i)^2}{Exp_i}$    


## Доверительные интервалы

#### Для отличий от среднего:  
Для метрик отношения: $p \pm Z^* \times \frac{s}{\sqrt{n}}$, где $Z^* = \frac{\hat{p} - p}{\sigma}$  
Для абсолютных метрик: $\bar{x} \pm t^* \times \frac{s}{\sqrt{n}}$, где $t^* = \frac{X - \mu}{\sigma}$, $df = min(n_1 - 1, n_2-1)$  
#### Для отличий двух значений:  
Для метрик отношения: $(\hat{p}_1 - \hat{p}_2) \pm Z^* \times \sigma_{\hat{p}_1 - \hat{p}_2}$, где:  
- $\sigma_{\hat{p}_1 - \hat{p}_2} = \sqrt{Var_1 + Var_2}$ = $\sqrt{\frac{\hat{p}_1 \times (1 - \hat{p}_1)}{n_1} + \frac{\hat{p}_2 \times (1 - \hat{p}_2)}{n_2}}$ 
- $Z^*$  для двухстороннего интервала

Для абсолютных метрик: $\bar{x} +- t^* \times \frac{s}{\sqrt{n}}$, где:  
- $t^* = \frac{\hat{X}_1 - \hat{X}_2}{\sqrt{\frac{s^2_1}{n_1} + \frac{s^2_2}{n_2}}}$
- $df = min(n_1 - 1, n_2-1)$
